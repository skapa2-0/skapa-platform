Retour au support

Pour obtenir une clé API, rendez-vous dans votre tableau de bord et cliquez sur "Générer une nouvelle clé API". Votre compte doit être approuvé par un administrateur avant de pouvoir utiliser les services.

Consultez notre documentation complète à l'adresse /api-docs/ pour découvrir tous les endpoints disponibles et leurs paramètres.

Vérifiez que :
  • Votre clé API est active
  • Vous incluez l'en-tête Authorization correctly
  • Votre compte est approuvé
  • Vous n'avez pas dépassé les limites de taux

Vous pouvez créer un ticket de support directement depuis cette interface. Pour des problèmes urgents, contactez-nous à support@intra-tech.fr.
